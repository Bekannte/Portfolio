MELODRAMA
I dissect myself with the experience of my youth, trying to find the perfect place in a repeated encounter with myself. The various different emotions of adolescence are presented in the form of a code merge, so that everyone can find resonance in their emotions.
Project Details / Background
This project combines images and code nicely, making them pixelated. The viewer does not at first think that the code is generated based on images and they have a sense of power. Understanding what each window represents based on the text makes the experience even more powerful. Although I encountered many problems in my attempts and it was still difficult to write a complete set of code on my own, I was largely able to solve it myself by referring to other people's code and reviewing my previous knowledge. However, the problem I haven't solved is the inability to make more variations with sound, which was my initial thought, and I hope that through my second semester I will be able to solve this problem and perfect this work.
